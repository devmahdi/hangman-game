# 🎮 Hangman Game

A simple, single-page hangman game built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies—just pure web fundamentals.

## 🎯 Play the Game

**[Play Now →](https://devmahdi.github.io/hangman-game/hangman.html)**

## ✨ Features

- **Single-page design** - Complete game in one HTML file
- **26 alphabet buttons** (A-Z) for letter selection
- **ASCII art hangman visual** that progressively draws with each wrong guess
- **Word display** showing underscores for hidden letters
- **25 programming-themed words** (JAVASCRIPT, ALGORITHM, DEBUGGING, etc.)
- **Win/loss detection** with clear status messages
- **Reset button** to start a new game anytime
- **Responsive design** with a beautiful purple gradient background
- **Color-coded feedback** - buttons turn green for correct guesses, red for wrong ones
- **6 wrong guesses allowed** before game over

## 🎨 Game Mechanics

- Guess letters by clicking the alphabet buttons
- Each correct guess reveals the letter in the word
- Each wrong guess adds a body part to the hangman
- Win by guessing the complete word before the hangman is fully drawn
- Lose after 6 wrong guesses
- Click "New Game" to play again
